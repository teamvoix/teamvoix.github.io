Будучи авторизованным Solution Provider GE Energy, мы предлагаем промышленные решения в сфере телекоммуникаций.

GE Smallworld PNI

SmallWorld Physical Network Inventory

Промышленное геоинформационное решение для учета и проектирования традиционных (ATM) и современных (IP/MPLS, FTTx, PON) сетей.

GE Smallworld LNI

Smallworld Logical Network Inventory

Комплексное моделирование и учет логических ресурсов для любых типов телекоммуникационных сетей (SDH, PDH).

GE Inventory Gateway

Smallworld Network Inventory Gateway

Решение для предоставления данных о всех физических и логических ресурсах во внешнюю или внутреннюю корпоративную сеть предприятия.
